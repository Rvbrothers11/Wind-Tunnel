# Wind-Tunnel
My project is a small portable wind tunnel for my hotwheels and airfoils at home. I am constantly building airfoils at home using wood, plastic, and other matirials, however, have been limited in how I can test them in terms of aerodynamics. IN COMES MY WIND TUNNEL!!! I built this wind tunnel to help me test my airfoils and small model cars and constantly adjust their flight dynamics and see their response with the help of... AIR!!!
